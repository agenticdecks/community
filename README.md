# Agentic Decks — Community

Welcome to the Agentic Decks community hub.

This is the place to ask questions, request features, report bugs, and inquire about paid plans.

## What is Agentic Decks?

[Agentic Decks](https://agenticdecks.com) builds rendering infrastructure for AI agents.
Our product, **Deckrun**, turns structured Markdown into polished PDFs, narrated videos, and audio assets — without leaving your AI system.

- **Free tier:** [free.agenticdecks.com](https://free.agenticdecks.com) — Markdown → PDF, no account required
- **Paid plans:** Full API with video, audio narration, connectors, and async job handling
